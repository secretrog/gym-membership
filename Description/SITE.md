# Site Vision: Gym Management System

A modern, dynamic membership and revenue management system for a local gym. It features a public-facing showcase and a private admin dashboard for managing members, revenue, and attendance.

## Core Features
1. **Public Showcase**: Photo gallery, schedules, and membership plans.
2. **Admin Dashboard**: Revenue tracking, member management, and bulk reminders.
3. **Attendance System**: QR code check-ins, workout streaks, and heatmaps.
4. **Member Experience**: Progress photo gallery and virtual tours.

## Design System Notes
- Use a premium, vibrant aesthetic (e.g., deep dark greens, energetic accents like neon yellow or orange).
- Emphasize modern typography and glassmorphism for admin cards.
- Ensure all views are responsive (Mobile-first for public, Desktop-first for Admin).
